💡 Electricity Narrative Visualization 💡

Overview


Welcome to my narrative visualization, implemented as an interactive web page, using my knowledge of visual communication and narrative structure, along with my skills of D3 programming in JavaScript. This is a publicly visible website.

URL of narrative visualization on Github : https://rawcdn.githack.com/SahaniGoonetilleke/CS416NarrativeVisualization/main/index.html

URL of essay : My essay describes how this interactive web page contains the elements of a narrative visualization along with requirements of the assignment.

My narrative visualization follows the "interactive slideshow" narrative visualization structure. User exploration is allowed at some or all of the steps of the story

💡 Electricity is a key necessities of the modern-world. Many would find it difficult to imagine a world without electricity. I have created this narrative visualization in an attempt to convey important data with regard to electricity utilization on a global scale.

Navigation buttons at the bottom of your screen will be visible and hovering in place for easy access throughout the visualization, irrespective of your screen size. Please click the "Next" button to begin.
